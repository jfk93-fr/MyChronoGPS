Pour installer les corrections
A partir du dossier /home/pi lancez les commandes suivantes :
	$ mkdir temp
	$ cd temp
	$ git clone https://github.com/jfk93-fr/MyChronoGPS.git
	$ cd MyChronoGPS
	$ sh correctif

Normalement, çà doit fonctionner tel quel
En cas de problème, lancer la procédure d'installation de MyChronoGPS
	$ cd $HOME/MyChronoGPS
	$ sh MyChronoGPS.sh

A la fin redémarrer le système
$ sudo reboot
